### Project Overview
This project focuses on analyzing crowdfunding campaign data to uncover key insights into what drives successful fundraising. The analysis is performed using SQL, Excel, Power BI, and Tableau, providing a complete end-to-end data analytics workflow.

### Objectives
Identify factors that influence campaign success

Analyze funding trends across categories and regions

Understand backer behavior and funding patterns

Build interactive dashboards for better decision-making

### Tools & Technologies
SQL – Data extraction & querying

Excel – Data cleaning & preprocessing

Power BI – Interactive dashboard creation

Tableau – Data visualization & storytelling

### Project Structure
crowdfunding.sql → SQL queries for data analysis

Excel_dashboard.jpg → Excel dashboard

PowerBI_dashboard.pdf → Power BI report

Tableau_dashboard.jpg → Tableau dashboard


### Key Insights
Campaigns with realistic funding goals have higher success rates

Categories like Technology & Design attract more funding

Early momentum (initial backers) plays a crucial role

Campaign duration impacts success probability
